How's it going Velt Developer Community!

Here's what shipped from `v4.5.6` until `v4.5.9-beta.4` across Core, Comments, Access Control, and Recorder.

:gear: *Core*

* Added `getHeartbeat()` method to track user-specific heartbeat data for any user, providing visibility into active sessions.
* Added `fetchDebugInfo()` and `getDebugInfo()` methods to retrieve debugging info about your Velt implementation.
* Optimized the initialization module and significantly reduced the SDK initialization time.

:speech_balloon: *Comments*

* Added preliminary infrastructure to capture screenshots in comments.
* Enhanced `selectCommentByAnnotationId()` to close selected comment annotations when called with no arguments or invalid ID.

:closed_lock_with_key: *Access Control*

* Launched Permission Provider to verify user access via your backend as source of truth. [Learn more](https://docs.velt.dev/key-concepts/overview#c-real-time-permission-provider).
* Added auto-revoke config to remove permissions when users log out or documents unset. [Learn more](https://docs.velt.dev/key-concepts/overview#c-real-time-permission-provider).
* Added Permission Provider events to monitor permission checks for debugging. [Learn more](https://docs.velt.dev/api-reference/sdk/models/data-models#permissionproviderevent).
* Added Access Context for granular feature-level permissions. [Learn more](https://docs.velt.dev/key-concepts/overview#set-feature-level-permissions-using-access-context-custom-metadata).
* Enforce Access Context restrictions preventing unauthorized users from adding comments, reactions, recorders, and area comments.

:movie_camera: *Recorder*

* Added video editor timeline image preview displaying frame snapshots for quick scene navigation.
* Fixed audio merging to combine microphone and tab audio in screen recordings.

For full API examples and details, check the [Velt SDK Changelog](https://docs.velt.dev/release-notes/version-4/sdk-changelog).

Happy shipping! :package:
