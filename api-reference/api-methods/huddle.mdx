


<Steps>
<Step title="enableCursorMode()">
This method enables cursor mode.
```jsx
const cursorSyncElement = client.getCursorSyncElement();
cursorSyncElement.enableCursorMode();
```
</Step>
<Step title="disableCursorMode()">
This method disables cursor mode.
```jsx
const cursorSyncElement = client.getCursorSyncElement();
cursorSyncElement.disableCursorMode();
```
</Step>
</Steps>
