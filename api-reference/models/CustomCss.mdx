## `CustomCss` Class

| Property | Type            | Optional | Description                  |
|----------|-----------------|----------|------------------------------|
| `type`   | 'link' \| 'styles' | No       | The type of custom CSS, either a link to a CSS file or inline styles. |
| `value`  | string          | No       | The value of the custom CSS, either a URL or CSS styles. |
