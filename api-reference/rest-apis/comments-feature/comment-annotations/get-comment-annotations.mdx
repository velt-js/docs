---
title: "Get Comment Annotations"
# api: "POST https://api.velt.dev/getcommentannotations"
api: "POST https://getcommentannotations.api.velt.dev/"
---


Use this API to retrieve comment annotations from a document within an organization.
Additional filters can be applied using location IDs, annotation IDs, or user IDs.

# Endpoint

`POST https://getcommentannotations.api.velt.dev/`

# Headers

<ParamField header="x-velt-api-key" type="string" required>
  Your API key.
</ParamField>

<ParamField header="x-velt-auth-token" type="string" required>
  Your [Auth Token](/security/auth-tokens).
</ParamField>

# Body Example

#### Params

<ParamField body="data" type="object" required>
  <Expandable title="properties">
    <ParamField body="organizationId" type="string" required>
      Organization ID
    </ParamField>

    <ParamField body="documentId" type="string" required>
      Document ID
    </ParamField>

    <ParamField body="locationIds" type="string[]">
      Array of Location IDs
    </ParamField>

    <ParamField body="annotationIds" type="string[]">
      Array of Annotation IDs
    </ParamField>

    <ParamField body="userIds" type="string[]">
      Array of User IDs
    </ParamField>

    <ParamField body="pageSize" type="number">
       Number of items to be retrieved per page (Optional). Default: 1000. 
     </ParamField>

     <ParamField body="pageToken" type="string">
       Page token retrieved from previous API call. (Optional)
     </ParamField>
    <ParamField body="order" type="string">
      Order of results based on lastUpdated timestamp (Optional). Options: 'asc' or 'desc'. Default: 'desc'.
    </ParamField>

    <ParamField body="statusId" type="string">
      Status ID to filter annotations on. (Optional)
    </ParamField>
  </Expandable>
</ParamField>

## **Example Requests**

```JSON

    "organizationId": "yourOrganizationId",

}

    ]
  }
}
```

# Response

#### Success Response

```JSON
{
  "result": {
    "status": "success",
    "message": "Annotations fetched successfully.",
    "data": [
      {
        "annotationId": "yourAnnotationId",
        "comments": [
          {
            "commentHtml": "<div>Hello</div>",
            "commentId": 123456,
            "commentText": "Comment text",
            "from": {
              "userId": "userId"
            },
            "status": "added",
            "type": "text"
          }
        ],
        "from": {
          "userId": "userId"
        },
        "type": "comment",
      },
      null // If annotation is not found
    ]
  }
}
```


#### Failure Response

```JSON
{
  "error": {
    "message": "ERROR_MESSAGE",
    "status": "INVALID_ARGUMENT"
  }
}
```
