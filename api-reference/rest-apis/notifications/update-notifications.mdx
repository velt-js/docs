Use this API to update notifications.

# Endpoint

`POST https://api.velt.dev/v1/notifications/update`

<Warning>
Known issue: This API is not working as expected. We are aware of the issue and are working on it.
</Warning>

</ParamField>

</ParamField>

## **Example Requests**
#### 1. Update by organizationId and documentId
```JSON
{
   "data": {
       "organizationId": "yourOrganizationId",
       "documentId": "yourDocumentId",
       "notifications": [
           {
               "id": "yourNotificationId",
               "displayBodyMessage": "This is body message (Secondary message)",
           }
       ]
   }
}
```

       "notifications": [
           {
               "id": "yourNotificationId",

# Response

#### Success Response

```JSON
{
   "result": {
       "status": "success",
       "message": "Notification(s) updated successfully.",
       "data": {
           "5471488637912692": {
               "success": true,
               "message": "Notification updated."
           }
       }
   }
}
```

#### When some notifications are not found

```JSON
{
   "result": {
       "status": "success",
       "message": "Notification(s) updated successfully.",
       "data": {
           "5471488637912692": {
               "success": false,
               "message": "Failed to update notification."
           },
           "5471488637912693": {
               "success": true,
               "message": "Notification updated."
           }
       }
   }
}
```

#### Failure Response

```JSON
{
  "error": {
    "message": "ERROR_MESSAGE",
    "status": "INVALID_ARGUMENT"
  }
}
```
