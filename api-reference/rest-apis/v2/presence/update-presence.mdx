---
title: "Update Presence"
api: "POST https://api.velt.dev/v2/presence/update"
---

Use this API to update presence data for existing users on a document within an organization.

# Endpoint

`POST https://api.velt.dev/v2/presence/update`

# Headers

<ParamField header="x-velt-api-key" type="string" required>
  Your API key.
</ParamField>

<ParamField header="x-velt-auth-token" type="string" required>
  Your [Auth Token](/security/auth-tokens).
</ParamField>

# Body

#### Params

<ParamField body="data" type="object" required>
  <Expandable title="properties">
    <ParamField body="organizationId" type="string" required>
      Organization ID
    </ParamField>

    <ParamField body="documentId" type="string" required>
      Document ID
    </ParamField>

    <ParamField body="users" type="object[]" required>
      Array of user objects to update. Each must include `userId`.
      <Expandable title="properties">
        <ParamField body="userId" type="string" required>
          Unique identifier for the user to update.
        </ParamField>

        <ParamField body="name" type="string">
          Updated display name of the user.
        </ParamField>

        <ParamField body="email" type="string">
          Updated email address of the user.
        </ParamField>

        <ParamField body="status" type="string">
          Updated presence status of the user (e.g., `"online"`, `"away"`, `"offline"`).
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

## **Example Requests**

#### Update a user's presence status

```JSON
{
  "data": {
    "organizationId": "org-123",
    "documentId": "doc-456",
    "users": [
      {
        "userId": "user-1",
        "status": "away"
      }
    ]
  }
}
```

#### Update multiple users

```JSON
{
  "data": {
    "organizationId": "org-123",
    "documentId": "doc-456",
    "users": [
      {
        "userId": "user-1",
        "status": "online",
        "name": "Updated Name"
      },
      {
        "userId": "user-2",
        "status": "away"
      }
    ]
  }
}
```

# Response

#### Success Response

```JSON
{
  "result": {
    "status": "success",
    "message": "Presence updated successfully.",
    "data": {
      "success": true
    }
  }
}
```

{

    "message": "ERROR_MESSAGE",
