---
title: "Get API Key Metadata"
api: "POST https://api.velt.dev/v2/workspace/apikeymetadata/get"
---

Use this API to retrieve metadata associated with a workspace API key.

<Info>
This endpoint uses **API-key-level auth**: pass `x-velt-api-key` and `x-velt-auth-token` as headers. You can obtain these from the [Get Auth Tokens](/api-reference/rest-apis/v2/workspace/authtokens-get) endpoint.
</Info>

# Endpoint

`POST https://api.velt.dev/v2/workspace/apikeymetadata/get`

# Headers

<ParamField header="x-velt-api-key" type="string" required>
  Your API key.
</ParamField>

<ParamField header="x-velt-auth-token" type="string" required>
  Your [Auth Token](/security/auth-tokens).
</ParamField>

# Body

This endpoint does not require a request body.

# Example Request

```JSON
{
  "data": {}
}
```

# Example Response

#### Success Response

```JSON
{
  "result": {
    "status": "success",
    "message": "API key metadata retrieved successfully.",
    "data": {
      "defaultDocumentAccessType": "public",
      "planInfo": {
        "type": "sdk test"
      },
      "ownerEmail": "owner@example.com"
    }
  }
}
```

#### Failure Response

```JSON
{
  "error": {
    "status": "PERMISSION_DENIED",
    "message": "Invalid API key."
  }
}
```
