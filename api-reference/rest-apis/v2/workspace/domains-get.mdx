---
title: "Get Domains"
api: "POST https://api.velt.dev/v2/workspace/domains/get"
---

Use this API to retrieve the list of allowed domains for a workspace.

# Endpoint

`POST https://api.velt.dev/v2/workspace/domains/get`

# Headers

<ParamField header="x-velt-api-key" type="string" required>
  Your API key.
</ParamField>

<ParamField header="x-velt-auth-token" type="string" required>
  Your [Auth Token](/security/auth-tokens).
</ParamField>

# Body

This endpoint does not require a request body.

# Example Request

```JSON
{
  "data": {}
}
```

# Example Response

#### Success Response

```JSON
{
  "result": {
    "status": "success",
    "message": "Allowed domains retrieved successfully.",
    "data": {
      "allowedDomains": [
        "localhost",
        "127.0.0.1",
        "example.com"
      ]
    }
  }
}
```

#### Failure Response

```JSON
{
  "error": {
    "status": "PERMISSION_DENIED",
    "message": "Invalid API key."
  }
}
```
