---
title: "Get Requested Domains"
api: "POST https://api.velt.dev/v2/workspace/requested/domains/get"
---

Use this API to retrieve all pending domain requests for a workspace.

# Endpoint

`POST https://api.velt.dev/v2/workspace/requested/domains/get`

# Headers

<ParamField header="x-velt-api-key" type="string" required>
  Your API key.
</ParamField>

<ParamField header="x-velt-auth-token" type="string" required>
  Your [Auth Token](/security/auth-tokens).
</ParamField>

# Body

This endpoint does not require a request body.

# Example Request

```JSON
{
  "data": {}
}
```

# Example Response

#### Success Response

```JSON
{
  "result": {
    "status": "success",
    "message": "Requested domains retrieved successfully.",
    "data": {
      "requestedDomains": {
        "req_abc123": {
          "clientDocumentId": "doc_abc123",
          "domain": "https://www.example.com",
          "documentName": "My Document",
          "id": "req_abc123",
          "status": "pending"
        }
      }
    }
  }
}
```

#### Failure Response

```JSON
{
  "error": {
    "status": "PERMISSION_DENIED",
    "message": "Invalid API key."
  }
}
```
