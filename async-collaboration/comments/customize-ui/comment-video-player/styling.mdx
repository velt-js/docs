## Disable ShadowDOM
By default, a ShadowDOM is used with certain components to ensure that your applicationâ€™s CSS does not interfere with the styling of the SDK components.

`Default: enabled`

If you want to customize the video player component using custom CSS, you can do so by disabling the ShadowDOM.


### Example
<Tabs>

