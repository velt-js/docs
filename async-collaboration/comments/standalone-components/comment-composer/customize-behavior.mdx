## context
Pass custom context data to comments created via the Comment Composer. The provided context object will be added to the comment annotation, allowing you to associate additional metadata with comments.

<Tabs>
<Tab title="React / Next.js">
```jsx
<VeltCommentComposer context={{ "key": "value" }} />
```
</Tab>

<Tab title="Other Frameworks">
```html
<velt-comment-composer context='{"key": "value"}'></velt-comment-composer>
```
</Tab>
</Tabs>

## submitComment
Programmatically submit a comment from a composer. Refer to [`submitComment()`](/async-collaboration/comments/customize-behavior#submitcomment) for more details.

## targetElementId
Set a unique identifier for this composer instance. Required when using programmatic submission via [`submitComment()`](/async-collaboration/comments/customize-behavior#submitcomment).

<Tabs>
<Tab title="React / Next.js">
```jsx
import { useCommentUtils } from '@veltdev/react';

function CommentSection() {
  const commentElement = useCommentUtils();

  const handleSubmit = () => {
    commentElement.submitComment('composer-1');
  };

  return (
    <>
      <VeltCommentComposer targetElementId="composer-1" />
      <button onClick={handleSubmit}>Submit Comment</button>
    </>
  );
}
```
</Tab>
<Tab title="Other Frameworks">
```html
<velt-comment-composer target-element-id="composer-1"></velt-comment-composer>
<button onclick="submitComment('composer-1')">Submit Comment</button>

<script>
  function submitComment(targetElementId) {
    const commentElement = Velt.getCommentElement();
    commentElement.submitComment(targetElementId);
  }
</script>
```
</Tab>
</Tabs>
