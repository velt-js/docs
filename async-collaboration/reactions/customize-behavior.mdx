## 1. Enable/Disable Dark Mode

You can enable or disable dark mode for the Inline Reactions feature.

`Default: false`

<Tabs>
<Tab title="React / Next.js">

```jsx
<VeltReactions darkMode={true} />
```

API Methods:
```jsx
const reactionElement = client.getReactionElement();
// To enable dark mode for standalone reaction feature
reactionElement.enableDarkMode();
// To disable dark mode for standalone reaction feature
reactionElement.disableDarkMode();
```
</Tab>

<Tab title="HTML">

```html
<velt-reactions dark-mode="true"></velt-reactions>
```

API Methods:
```jsx
const reactionElement = Velt.getReactionElement();
// To enable dark mode for standalone reaction feature
reactionElement.enableDarkMode();
// To disable dark mode for standalone reaction feature
reactionElement.disableDarkMode();
```
</Tab>
</Tabs>

## 2. Custom Reactions

You can set custom reactions by passing a map that contains information about the reactions you want to add.

The map keys should be the reaction ID, and the map value should contain an object with either a `url`, `svg`, or `emoji` field to represent the reaction icon you want to use.

<Tabs>
<Tab title="React / Next.js">

```jsx
const customReactions = {
    "URL_EMOJI_ID": {
        "url": "EMOJI_URL"
    },
    "SVG_EMOJI_ID": {
        "svg": "EMOJI_SVG"
    },
    "TEXT_EMOJI_ID": {
        "emoji": "ðŸ¤£" // emoji as a text
    }
};

<VeltReactions customReactions={customReactions} />
```

API Methods:
```jsx
const reactionElement = client.getReactionElement();
reactionElement.setCustomReactions(customReactions);
```
</Tab>

<Tab title="HTML">

```html
<velt-reactions
  custom-reactions='{"fire_url":{"url":"https://em-content.zobj.net/source/apple/391/fire_1f525.png"},"svg_face":{"svg":"EMOJI_SVG"},"text_emoji":{"emoji":"ðŸ¤£"}}'>
</velt-reactions>
```

API Methods:
```jsx
const reactionElement = Velt.getReactionElement();
const customReactions = {
    "URL_EMOJI_ID": {
        "url": "EMOJI_URL"
    },
    "SVG_EMOJI_ID": {
        "svg": "EMOJI_SVG"
    },
    "TEXT_EMOJI_ID": {
        "emoji": "ðŸ¤£" // emoji as a text
    }
};
reactionElement.setCustomReactions(customReactions);
```
</Tab>
</Tabs>

## 3. Enable/Disable Reactions

You can enable or disable the Reactions feature programmatically.

<Tabs>
<Tab title="React / Next.js">

API Methods:
```jsx
const reactionElement = client.getReactionElement();
// To enable reactions
reactionElement.enableReactions();
// To disable reactions
reactionElement.disableReactions();
```
</Tab>

<Tab title="HTML">

API Methods:
```jsx
const reactionElement = Velt.getReactionElement();
// To enable reactions
reactionElement.enableReactions();
// To disable reactions
reactionElement.disableReactions();
```
</Tab>
</Tabs>
