---
title: "Recorder"
sidebarTitle: "Overview"
description: "The Recorder allows your users to create audio, screen, and video recordings all with the click of a button."
---



The Velt SDK contains 4 components that can be used to control the `Recorder` functionality:
 - `VeltRecorderNotes` - this is used to enable users to pin `Recorder Notes` on the screen similar to pinning Comments.
 - `VeltRecorderTool` - this is the button that is used to trigger a recording and pin a `Recorder Note`
 - `VeltRecorderControlPanel` - this is a control panel that can be used to start, stop, and pause an active recording
 - `VeltRecorderPlayer` - this is a component that allows you to play previous recordings by their ID
