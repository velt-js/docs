<Step title="Get the Velt client">
Import the `useVeltClient` React hook.

You can use this hook within your component to fetch the Velt client.

```js
import { useVeltClient } from '@veltdev/react';

const { client } = useVeltClient();
```

</Step>

<Step title="Create a useEffect hook">
Create the hook with the client as dependency.

```js
useEffect(() => {
    if (client) {
      // ...
    }
  }, [client]);
```

</Step>

<Step title="Enable live selection">
When `Live Selection` is enabled, users will be able to see a highlighted box around HTML elements that other users are currently interacting with.

`Live Selection` works out of the box with Text nodes, Input tags & Contenteditable divs.

You can enable it with:

```js
const selectionElement = client.getSelectionElement();
selectionElement.enableLiveSelection();
selectionElement.disableLiveSelection(); // to disable
```

</Step>

<Step title="Initialize with your API Key">
Put this in your root app script:
```js
async function loadVelt() {
    await Velt.init("YOUR_VELT_API_KEY");
}
```
</Step>

<Step title="Enable live selection">
When `Live Selection` is enabled, users will be able to see a highlighted box around HTML elements that other users are currently interacting with.

`Live Selection` works out of the box with Text nodes, Input tags & Contenteditable divs.

You can enable it with:

```js
if(Velt) {
  const selectionElement = Velt.getSelectionElement();
  selectionElement.enableLiveSelection();
  selectionElement.disableLiveSelection(); // to disable
}
```

```js React / Next.js
import { useVeltClient } from '@veltdev/react';
import { useEffect } from 'react';

export default function App() {

  const { client } = useVeltClient();

  useEffect(() => {
    if (client) {
      const selectionElement = client.getSelectionElement();
      selectionElement.enableLiveSelection(); //enable live selection
      selectionElement.disableLiveSelection(); //disable live selection
    }
  }, [client]);

  return (
  );
}
```
