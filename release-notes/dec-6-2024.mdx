# 3.0.78

### Features
- [**Velt Button**]: Added Velt Button Wireframe Component with support for various types ('button', 'button-toggle', 'multi-select', 'single-select'), disabled state, active state, and group functionality. Available in React and HTML with corresponding event handling APIs.
- [**Theme Playground**]: Added a new [Theme Playground](https://playground.velt.dev/themes) to customize and test your themes. It will generate CSS variables that you can just copy and paste into your app.
