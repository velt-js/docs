## Added "velt-comment-pin" Component 
We have introduced the velt-comment-pin component. You can use the Comment Pin component to manually set the position of a Comment Annotation.

<Tabs>
<Tab title="React / Next.js" >
```jsx

```
</Tab>

</Tab>
