## Overview

The Activity Log Primitives API provides granular components that can be used independently to build fully custom activity log interfaces. Each primitive accepts `defaultCondition` to control its render visibility.

## Common Inputs

All Activity Log primitive components accept the following input.

| React Prop | HTML Attribute | Type | Default | Description |
|------------|----------------|------|---------|-------------|
| `defaultCondition` | `default-condition` | `boolean` | `true` | When `false`, the component always renders regardless of internal state. |
