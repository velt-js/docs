---
title: Comment Bubble Primitives
description: Primitive-based components for building custom Comment Bubble interfaces with maximum flexibility.
---

These components are HTML-only in this release. No React wrapper components are available.

### velt-comment-bubble-avatar

Avatar displayed inside the comment bubble.

```html
<velt-comment-bubble-avatar annotation-id="abc123"></velt-comment-bubble-avatar>
```

**Attributes:** Common attributes only (see [Common Inputs](#common-inputs) section).

---

### velt-comment-bubble-comments-count

Comment count displayed inside the comment bubble.

```html
<velt-comment-bubble-comments-count annotation-id="abc123"></velt-comment-bubble-comments-count>
```

**Attributes:** Common attributes only (see [Common Inputs](#common-inputs) section).

---

### velt-comment-bubble-unread-icon

Unread indicator icon displayed inside the comment bubble.

```html
<velt-comment-bubble-unread-icon annotation-id="abc123"></velt-comment-bubble-unread-icon>
```

**Attributes:** Common attributes only (see [Common Inputs](#common-inputs) section).
