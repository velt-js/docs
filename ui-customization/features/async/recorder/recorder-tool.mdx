# Props

## maxLength
Set a maximum recording duration limit in seconds. When the limit is reached, recording will automatically stop.

`Default: undefined` (no limit)

<Tabs>
<Tab title="React / Next.js">
```jsx
import { VeltRecorderTool } from '@veltdev/react';

function YourComponent() {
  return (
    <VeltRecorderTool maxLength={300} /> // 300 seconds limit
  );
}

// Using the useRecorderUtils hook
import { useRecorderUtils } from '@veltdev/react';

function YourComponent() {
  const recorderUtils = useRecorderUtils();

  useEffect(() => {
    // Set max length programmatically
    recorderUtils.setMaxLength(300); // 300 seconds
  }, []);

  return <VeltRecorderTool />;
}
```
</Tab>
<Tab title="Other Frameworks">
```html
<!-- Set max length using HTML attribute -->
<velt-recorder-tool max-length="300"></velt-recorder-tool>

<script>
// Set max length programmatically
const veltClient = await Velt.init("YOUR_VELT_API_KEY");
const recorderElement = veltClient.getRecorderElement();
recorderElement.setMaxLength(300); // 300 seconds
</script>
```
</Tab>
</Tabs>

## pictureInPicture
Enable Picture-in-Picture mode for screen recordings with camera. This allows users to view the recording in a floating window.

`Default: false`

<Warning>
Picture-in-Picture is only supported in Chrome browsers and only works for screen recordings when the camera is active. This feature is disabled by default.
</Warning>

<Tabs>
<Tab title="React / Next.js">
```jsx
import { VeltRecorderTool } from '@veltdev/react';

function YourComponent() {
  return (
    <VeltRecorderTool pictureInPicture={true} />
  );
}

// Using the useRecorderUtils hook
import { useRecorderUtils } from '@veltdev/react';

function YourComponent() {
  const recorderUtils = useRecorderUtils();

  useEffect(() => {
    // Enable Picture-in-Picture programmatically
    recorderUtils.enablePictureInPicture(true);
  }, []);

  return <VeltRecorderTool />;
}
```
</Tab>
<Tab title="Other Frameworks">
```html
<!-- Enable Picture-in-Picture using HTML attribute -->
<velt-recorder-tool picture-in-picture="true"></velt-recorder-tool>

<script>
// Enable Picture-in-Picture programmatically
const veltClient = await Velt.init("YOUR_VELT_API_KEY");
const recorderElement = veltClient.getRecorderElement();
recorderElement.enablePictureInPicture(true);
</script>
```
</Tab>
</Tabs>
